# Twitch Chat Overlay (Simulação)

Projeto de overlay de chat inspirado na Twitch, desenvolvido para estudos de HTML, CSS e JavaScript.

## 📺 Sobre o projeto

Este projeto simula um chat ao vivo com mensagens automáticas, podendo ser usado como base para overlays em lives.

## ⚙️ Funcionalidades

- Simulação de mensagens em tempo real
- Nomes aleatórios de usuários
- Cores estilo Twitch
- Layout transparente para OBS

## 🚀 Como usar

1. Abra o arquivo `index.html`
2. Adicione no OBS como:
   - Browser Source
3. Ajuste tamanho e posição na tela

## ⚠️ Observação

Este projeto é apenas uma simulação.  
Não está conectado ao chat real da Twitch.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Praticar manipulação de DOM e criação de interfaces dinâmicas com JavaScript.
